dn->dnseq

(dn->dnseq dn)

Returns a sequence of the sub-DNs of dn

(dn->dnseq "dc=example,dc=co,dc=uk")
;=> ["dc=uk" "dc=co,dc=uk" "dc=example,dc=co,dc=uk"]

Each DN in the sequence is an instance of the Unboundid SDK DN class

view source

dn-is-missing?

(dn-is-missing? c dn)

Returns try if a get for the specified DN succeeds, false otherwise. Note that domain DNs are always assumed to exist, no gets are performed for them.

rename-dn

(rename-dn c dn new-dn)

Renames dn to new-dn in the directory. new-dn can either be a partial or full DN to optionally move/re-base the object within the directory as well as renaming it.

Throws an LDAPException on error
